# HUD CURVE (RedM Radar UI)

**Description**
HUD Curve is a curved radar HUD designed for RedM.  
It reads player stats from metabolism_pro and displays icons around the minimap.

Features:
- Curved minimap icon layout
- Stress brain icon with animation
- Food / Water / Temperature indicators
- Auto radar alignment
- Lightweight draw system
- Compatible with Metabolism Pro export

Requirements:
metabolism_pro resource must be running.

---

**คำอธิบาย**
HUD Curve คือระบบ HUD แบบโค้งรอบ minimap สำหรับ RedM  
ใช้แสดงค่า stress อาหาร น้ำ และอุณหภูมิ โดยอ่านค่าจาก metabolism_pro

ความสามารถ:
- ไอคอนเรียงโค้งรอบเรดาร์
- ไอคอนสมองแสดงความเครียด
- แสดงค่าอาหาร น้ำ และอุณหภูมิ
- จัดตำแหน่งตาม radar อัตโนมัติ
- ระบบวาด UI เบา ไม่กินเฟรม
- ทำงานร่วมกับ metabolism_pro

สิ่งที่ต้องมี:
ต้องเปิด resource metabolism_pro ก่อน
